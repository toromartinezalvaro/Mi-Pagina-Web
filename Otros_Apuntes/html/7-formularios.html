COMO HACER FORMULARIOS "< form>< /form>"

Los formularios son una parte de las paginas web con las que los usuarios pueden interactuar ya sea escribiendo texto o haciendo alguna accion que despues seran procesados los datos para que el programa de una respuesta.


Atributos y metodos de los formularios:

los atributos y metodos van dentro de la primera etiqueta "form" asi "< form atributo="" method="">< /form>". Existen varios atributos para los formularios, aqui esta la descripcion y uso de algunos.

  • action="" :Este atributo recibe como valor una "url" que se encarga de procesar los datos del formulario, como por ejemplo un archivo "datos.php" de la siguiente forma

    • "< form action="datos.php">< /form>"

  • method="" :Al usar el atributo "action" debemos usar el atributo "method" que nos servira para determinar por cual metodo vamos a enviar nuestra informacion, sea el metodo "POST" o el metodo "GET" de la siguiente manera:

    • < form action="" method="POST">< /form>
    • < form action="" method="GET">< /form>

COMO HACER UN INPUT:

Ahora veremos los tipicos campos de texto en los que siempre nos piden por ejemplo el nombre, el correo, contraseña etc. TODOS LOS CAMPOS SE LLAMARAN CON LA ETIQUETA "input" que puede contener diversidad de atributos que nos ayudaran a determinar el funcionamiento y visualizacion de nuestra etiqueta (recordando que lo siguiente ya estaria dentro de la etiqueta "form"). Los atributos serian los siguientes:


ATRIBUTOS DE LOS INPUT:

  • TYPE:El atributo "type" nos sirve para especificar que tipo de dato se usara en el "input", EJEMPLO:

    < input type="text"> Esta linea de codigo generaria el siguiente "input" en el que podremos escribir nuestro nombre

    Tambien podemos usar en el atributo "type" un "password" en el que no se vera lo que escribimos de la siguiente manera

      < input type="password">


    Si usamos el atributo type="email", tendremos un input que verificara que el correo que escribamos en el sea correcto, pero debemos. EJEMPLO:

      < input type="email" name="" id="">



    Otro atributo "type" que podemos usar el es "checkbox" que nos permitira hacer un recuadro de chequeo para elegir alguna opcion. si deseamos que solo se pueda elegir una opcion usaremos type="radio" EJEMPLO:

    • < input type="checkbox">HOMBRE
    • <|input type="checkbox">MUJER

      • HOMBRE MUJER
    • < input type="radio">HOMBRE
    • <|input type="radio">MUJER

      • HOMBRE MUJER


  • VALUE: El atributo "value" sirve para dar un valor al "input", pero cuando el usuario quiera por ejemplo escribir su nombre, tendra que borrar el mensaje que pongamos en el atributo "value", en este caso "Escribe tu nombre". EJEMPLO (intenta escribir tu nombre):

    < input type="text" value="Escribe tu nombre">



  • PLACEHOLDER: El atributo "placeholder" sirve para dar un valor al "input", pero cuando el usuario quiera por ejemplo escribir su nombre NO tendra que borrar el mensaje que pongamos en el "input". EJEMPLO(intenta escribir tu nombre):

    < input type="text" placeholder="Escribe tu nombre">



  • SIZE :En este ultimo "input" agregamos el atributo size que nos permitira modificar el tamaño del mismo. por ejemplo en este caso, el input sera mas pequeño que los demas ya que el valor de su atributo "size" es menor al de los demas. A pesar de servir para modificar el tamaño del "input", no se aconseja usarlo ya que es mas practico modificarlo con "css". Igualmente el ejemplo es el siguiente:

    < input type="text" placeholder="Escribe tu nombre" size="10"> y daria como resultado el siguiente "input"



  • :El atributo "maxlength" nos servira para determinar el numero de caracteres que permitira ingresar nuestro "input", por ejemplo si ponemos "7" en el atributo "maxlength" solo podremos escribir 7 caracteres, intenta escribir en el ejemplo mas de 7 caracteres

    < input type="text" placeholder="escribe tu nombre" maxlength="7">




  • NAME: El atributo "name" se puede usar en cualquier elemento de nuestro formulario y nos permite indicarle al servidor que valor estamos poniendo a nuestro "input" asi podremos identificar direfentes "inputs" segun su nombre (name)

    < input type="text" placeholder="escribe tu nombre" name="">



  • ID: El atributo "id" nos sirve para poder identificar el "input", lo que nos permitira usar dicha identificacion para por ejemplo con el uso de una etiqueta "label" (añadiendo el id a su atributo "for"), podremos añadir un texto en el cual al darle click, pondra la barra de escrituta en nuestro input ejemplo:

    < label for="prueba">Escribe tu nombre:

    < input type="text" placeholder="Escribe tu nombre" id="prueba">

    las dos anteriores lineas de codigo estan conectadas por el atributo "for" de la etiqueta "label" y el atributo "id" de la etiqueta "input" por lo cual al dar click en el mensaje de la etiqueta "label" (que en este caso seria "Escribe tu nombre:") podremos empezar a escribir en el cuadro de texto(input).
