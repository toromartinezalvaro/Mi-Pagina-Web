LISTAS ORDENADAS Y DESORDENADAS


Listas ordenadas y desordenadas:

tienen 2 funciones, poner un contenido que queremos compilar para mostrar de una manera mas facil, como tambien utilizar esas listas para con "css" y otros lenguajes darle un formato y hacer que sea un menu de navegacion o algun otro elemento


LISTAS DESORDENADAS "< ul >< /ul >":

la etiqueta ul sirve para hacer listas desordenadas (sin un orden especifico). Dentro de la etiqueta "ul" podemos usar la etiqueta "li" (< li>< /li >) para poner diferentes items. Cada "li" representa un item


Ademas podemos hacer que cada "li" sea un link agregando la etiqueta "a" dentro de la misma un ejemplo de como hacerlo seria el siguiente: "< li>< a href="#"> item numeroX < /a>< /li>""

EJEMPLO DEL RESULTADO:

  • item 1
  • item 2
  • item 3
    • item #1
    • item #2
  • item 4
  • item 5


LISTAS ORDENADAS "< ol>< /ol>"

Las listas desordenadas se hacen con la etiqueta "ol" y funcionan igual que las listas desordenadas, la diferencia esta en que en este dipo de lista, los items tendran un orden que podria ser por ejemplo numeracion (en los siguientes ejemplo no usare la etiqueta "< a href="">< /a>" que sirve para agregar un link a los items).


  1. item #1
  2. item #2
  3. item #3
    • item #1
    • item #2
  4. item #4
  5. item #5
