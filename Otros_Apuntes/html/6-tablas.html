TABLAS (< table >< /table >)

las tablas hoy en dia son algo polemicas, han dado problemas a lo largo del tiempo, se usan principalmente en html para tabular contenidos(una tabla con datos para mostrar estadisticas, valores,etc). Las tablas no deben ser utulizadas para estructurar un sitio web


Para hacer tablas usamos la etiqueta "table" y dentro de ella pondremos etiquetas que permitiran hacer filas y/o columnas.


PARA HACER FILAS(horizontal) "tr":

Para hacer filas usamos la etiqueta "< tr>< /tr>", cada etiqueta "tr" sera equivalente a una fila, por lo que si queremos una tabla de 4 filas, deberemos hacer 4 etiquetas "tr" con sus etiquetas "td" correspondientes(las etiquetas "td" agregan el numero de columnas)


PARA HACER COLUMNAS(vertical) "td":

para hacer columnas vamos a usar la etiqueta "< td>< /td>" en donde agregaremos el numero de elementos


UN EJEMPLO DE LO ANTERIOR SERIA EL SIGUIENTE

Si por ejemplo escribieramos lo siguiente (son escribir los "------"):

< table>
----< tr>
--------< td>columna 1< /td>
--------< td>columna 2< /td>
-------- td>columna 3< /td>
----< /tr>
----< tr>
--------< td>columna 1< /td>
--------< td>columna 2< /td>
--------< td>columna 3< /td>
----< /tr>
----< tr>
--------< td>columna 1< /td>
--------< td>columna 2< /td>
--------< td>columna 3< /td>
----< /tr>
< /table>

obtendriamos una tabla de 3 filas y tres columnas como la siguiente:

columna 1 columna 2 columna 3
columna 1 columna 2 columna 3
columna 1 columna 2 columna 3
