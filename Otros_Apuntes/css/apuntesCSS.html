Â¿COMO FUNCIONA "CSS"?

"CSS" funciona con reglas que llevaran dentro de llaves {} las propiedades de la regla. Para entender mejor que es una regla mira el siguiente ejemplo del selector "body" que nos indica a que aspecto vamos a realizar el cambio(en este caso en el body) y dentro de este selector vamos a agregar la propiedad "background" que nos permitira cambiar el fondo de nuestro selector "body". A la propiedad "background" vamos a agregarle un valor (en este caso #000000 que es equivalente al color negro pero tambien podriamos reemplazar ese valor por "black")

  • body {background:#000000;}
