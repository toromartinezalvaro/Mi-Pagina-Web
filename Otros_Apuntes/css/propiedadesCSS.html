PROPIEDADES DE LOS SELECTORES EN "CSS"

Existen diferentes tipos de propiedades para modificar los selectores de "CSS", cuando escribimos una propiedad, debemos cerrar la orden con ";". Algunos selectores son los siguientes:

  • background: Sirve para modificar el color de fondo de nuestro selector, se puede usar por ejemplo "background:#000000;" o "background:black;" para poner el fondo de color negro

  • color: Sirve para cambiar el color de la fuente de letra se puede usar por ejemplo "color:#fff" para que la letra sea de color blanco

  • margin:0; Nos permite agregar un margen, se haria por ejemplo de la siguiente forma "margin:0;"

  • padding:0; sirve para cambiar el margen interno que tienen los elementos

  • font-size:20px; se usa para cambiar el tamaño de la fuente, nos permite decir cuantos pixeles queremos que tenga

  • display:table; sirve para organizar el contenido en una barra que se auto ajusta en cuanto al tamaño, si por ejemplo tenemos una barra de busqueda que crece, automaticamente el resto de elementos disminuiran su tamaño
