Css significa Cascading Style Sheets (hoja de estilo en cascada) por que se ejecuta de arriba hacia abajo.
Existen 3 formas de trabajar con CSS:

  1. crear un nuevo documento y enlazarlo con html (esta es la mas recomendable)

  2. colocando la etiqueta Style debajo de la etiqueta de cierre head(< /head>) en el archivo de html(se usa para probar el css antes de crear su respectivo archivo).

  3. escribir dentro de los atributos (se usa el atributo style con la finalidad de modificar la etiqueta directamente)


Que son los selectores en CSS?

Los selectores en CSS se usan para indicar que etiquetas queremos modificar, podemos hacerlo haciendo uso del nombre de la etiqeta(lo cual modificara todas las etiquetas de ese tipo) o agregarle clases y/o id a las etiquetas y usar dichas clases como selectores. Si queremos hacer un selector de una clase, este debera llevar un punto(.) antes del nombre de la clase (si quiero aplicarle mas de una clase a una etiqueta, dentro de las comillas solo debo separar los nombres de las clases con espacios asi: < p class”clase_1 clase_2” > < /p>), si por el contrario el selector es para un atributo "id" este debella llevar el signo numeral(#) antes del nombre del selector, por ultimo, si queremos hacer un selector con una etiqueta solo usaremos el nombre de la etiqueta sin ningun signo previo. Luego de esto se deberan abrir y cerrar llaves y dentro de las mismas iran los atributos que deseamos modificar
Ejemplo:

  • Selector de tipo clase: class="perro" : .perro{atributos a cambiar aqui}
  • Selector de tipo id: id="perro" : #perro{atributos a cambiar aqui}
  • Selector de agrupacion: < p>< /p> : p{atributos a cambiar aqui}
  • Selector de tipo Universal (*) Modifica todas las caracteriaticas de todas las etiquetas

Cuales son los atributos de los selectores que existen en CSS?

  • color: red; sirve para cambiar el color de la fuente

  • font-size: 20px; sirve para dar el tamaño de la fuente en píxeles

  • text-align: center; sirve para alinear el texto, en este caso en el centro

  • width: 700px; modifica el ancho

  • height: 500px; modifica la altura

  • border: solid blue; hace un borde sólido de color azul

  • margin: 0 0 0 0; modifica el margen, el primer cero es la parte de encima, el segundo de la derecha, tercero de abajo y el último de la izquierda

  • type: Se usa para definir el tipo de dato que se ingresara en la etiqueta input, existen los siguientes tipos de datos para type:
    • text: para definir que sera un texto
    • password: contraseña
    • radius: muestra un circulo que se marca
    • date: coloca un calendario para seleccionar una fecha
    • email: requiere que se ingrese un @ por que es para un email
    • range: barra escalable (se puede modificar dicho rango)

  • font-family: verdana; sirve para cambiar el tipo de fuente de la letra como por ejemplo verdana o times new romans

  • border: para crear bordes
    • solid 1px red; crea un borde solido de 1 pixel de color rojo

  • padding: espacio entre el centro y el borde (por defecto el tamaño es cero)

  • border-radius redondea los bordes de un boton

  • cursor: pointer; cambia el estilo del cursor cuando se pone encima del boton

  • outline: none; para quitar lña linea azul que se hace cuando le doy click al boton
