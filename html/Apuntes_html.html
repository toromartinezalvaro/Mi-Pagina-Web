Etiquetas de HTML


  • title: muestra el título de la página en la barra de título del navegador

  • i: coloca el texto en italica(cusiva)

  • em: coloca el texto en italica (cursiva), Pero a diferencia de la etiqueta "i", la etiqueta "em" hace un enfasis en el contenido que contiene

  • strong: (énfasis fuerte) Para enfatizar considerablemente un texto (Nota: el texto aparecerá en negrita. También en este caso la negrita es un solo una consecuencia . El navegador eligió mostrar las palabras importantes en negrita para hacer que destaquen más.)

  • mark: (Resaltador) se usa para resaltar visualmente una parte del texto. El extracto no se tiene que considerar necesariamente importante, pero queréis que destaque del resto del texto.

  • ul: Sirve para hacer listas desordenadas(por puntos). delimita la lista completa

  • li: define un elemento de la lista (crea una viñeta en la lista, cada li es una viñeta para un item)

  • ol: Sirve para hacer listas ordenadas (por numeros). delimita la lista completa

  • a: Sirve para crear un enlace (el enlace se pondra en el atributo href="Enlace aqui")

  • div: division (para hacer cajas, divisiones, etc)

  • form: para hacer formularios se usa de forma similar a las etiquetas "ul" u "ol" (dentro de la etiqueta form se ponen etiquetas como por ejemplo "label", "input", "button", etc )

  • input: cpermite al usuario ingresar datos desde teclado o mouse

  • label: crea un texto antes del recuadro de input (se pueden enlazar) como por ejemplo escribir usuario o contraseña antes de los recuadros de usuario o contraseña respectivamente

  • iframe: sirve para poner un archivo html dentro de otro

  • button: Crea un boton

  • table: Sirve para crear tablas, dentro se deben usar las etiquetas tr(para las filas) y dentro de las etiquetas tr usar las etiquetas td(para las celdas) o th(para las celdas que contienen titulos)

  • th: Crea celdas con titulos (complementa la etiqueta table)

  • tr: crea cada fila de la tabla (complementa la etiqueta table)

  • td: crea una celda en la tabla (complementa la etiqueta table)

  • style: Sirve para agregar estilo css directamente en el archivo html (normalmente se usa entre el head y el body)












Como crear un enlace entre una pagina y otra:

Los enlaces se crearan mediante la etiqueta "a", esta etiqueta debera tener un atributo llamado "href" luego se pondra un signo "=" seguido de comillas que contendran el enlace al que deseamos ir
Ejemplo: < a href="google.com">enlace a google< /a>


Como hacer enlace a un hipervinculo de la misma pagina (para que se vaya directo a la parte que deseamos)

Un hipervínculo es una especie de punto de referencia que puedes incluir en tus páginas HTML cuando son muy largas.
De hecho, pueden ser útiles para crear un enlace a un punto situado más abajo en la misma página para permitir a los visitantes saltar directamente a la parte que les interesa.
Para crear un hipervínculo nuevo, Puede ser cualquier etiqueta, por ejemplo un título.
Para dar un nombre al hipervínculo, se utiliza el atributo id. Entonces podremos usarlo para crear un enlace a este hipervínculo.
Ahora solo tienes que crear un enlace como siempre, pero esta vez el atributo href incluirá una almohadilla (#) seguida del nombre del enlace.
Ejemplo:
Etiqueta numero 1:

probando enlace


Etiqueta numero 2: (esta nos llevara a la otra) este link nos dirigira a "probando enlace"


Como crear un enlace a un hipervínculo situado en otra página:

La idea es crear un enlace que abra otra página Y que te lleve directamente a un hipervínculo situado más abajo de esta página.
En la práctica es muy fácil hacerlo: escribe el nombre de la página, seguido por una almohadilla ( #), seguido del nombre del hipervínculo.
Ejemplo: < a href="anchors.html#rodillos">< /a>
El anterio ejemplo te llevará a la página anchors.html, directamente al hipervínculo llamado rodillos


Atributos:


Los atributos estan situados dentro de las etiquetas y sirven para establecer las caracteristicas de las mismas, los atributos se usan poniendo el nombre del atributo seguido de el signo igual(=) y finalmente entre comillas se pondra la identificacion del atributo
Ejemplo: atributo="aqui lo que lo identifica"


  • href: Se usa en las etiquetas "a", sirve para determinar el destino con el que vamos a enlazar dicha etiqueta

  • id: Se usa para dar una identificacion a la etiqueta, el id es unico y no se debe usar el mismo en diferentes etiquetas

  • title: muestra una descripción emergente cuando coloques el curso sobre el enlace. La descripción emergente puede ser útil para proporcionar información a los visitantes incluso antes de que hayan hecho clic sobre el enlace.

  • target="_blank": Obliga al enlace a abrirse en una nueva ventana

  • type: se usa en la etiqueta input





Como recibir correos?

para recibir correos usariamos la etiqueta "a" que nos llevara al link de nuestro correo y dentro de esta etiqueta pondremos el atributo "href" que contendra en principio el comando "mailto:" seguido de nuestro correo electronico
Ejemplo: < a href="mailto:torom.alvaro@gmail.com">Enviame un correo electronico!< /a>


Como descargar un archivo?

El procedimiento es exactamente el mismo que para enlazar a una página web, pero esta vez tenemos que indicar el nombre del archivo que queremos que se descargue.
Por ejemplo, imagina que quieres descargar "myfile.zip" Solo tienes que colocar este archivo en la misma carpeta que tu página web (o en una subcarpeta) y crear un enlace a este archivo.
Ejemplo: < a href="myfile.zip">Descargar el archivo< /a>
Cuando el navegador vea que no hay ninguna página web que mostrar, comenzará el proceso de descarga cuando el usuario haga clic sobre el enlace.
