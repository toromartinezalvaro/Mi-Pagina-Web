METER CODIGO DENTRO DE ETIQUETAS---HTML TAG wrapper:

"CTRL + i": Si tenemos codigo escrito y lo queremos meter dentro de por ejemplo una etiqueta "div", señalamos el codigo y seguido presionamos "CTRL"+"i"


alt + click

Si quisiera cambiar el texto de diferentes etiquetas a la vez, con esta extension se puede hacer, solo es necesario mantener presionado "alt" mientras se da click en los diferentes sitios a editar


< a href="">alt + click en cada uno de estos< /a>
< a href="">alt + click en cada uno de estos< /a>

P.HOLA es igual que < p class="HOLA">< /p >

"p.hola" es un ejemplo para aclarar como ahorrar tiempo al escribir codigo, si escribimos el nombre de una etiqueta, seguido de un punto y un string, se creara la etiqueta y el string sera la clase de esa etiqueta(el punto crea la clase de la etiqueta), por lo cual al escribir "p.hola" y dar enter, se crearia la siguiente linea de codigo(sin espacios) < p class="hola>< /p >"


PARA PONER UN "ID" A UNA ETIQUETA(div#nombre_id):

Usando una etiqueta "div"(por ejemplo), para agregarle un "id", escribiriamos por ejemplo "div" + "#" + "nombre del id" obteniendo la siguiente linea de codigo(sin espacios) < div class="" id="nombre del id">< /div >


COMO ESCRIBIR ETIQUETAS DENTRO DE OTRAS DE FORMA RAPIDA(header>nav>ul>li)

Si por ejemplo queremos crear una etiqueta "header" que contenga una etiqueta "nav" que a su vez contenga una etiqueta "ul" que a su vez contenga 5 etiquetas "li" usariamos el simbolo ">" para indicar cual etiqueta va dentro de cual lo hariamos escribiendo lo siguiente "header>nav>ul>li*5" y por ultimo dariamos enter para obtener el codigo

EJEMPLO (sin los puntos):
....< header>
........< nav>
............< ul>
................< li>< /li>
................< li>< /li>
................< li>< /li>
............< /ul>
........< /nav>
....< /header>


AGREGAR VARIAS ETIQUETAS SIN QUE UNA CONTENGA A LA OTRA(nav+p+a)

usariamos el simbolo "+" para que se ponga una etiqueta debajo de la otra por ejemplo si ponemos "nav+p+a" hara una etiqueta "nav", debajo hara una etiqueta "p" y por ultimo debajo hara una etiqueta "a" si quisieramos que estas etiquetas estuvieran dentro de la etiqueta "header"por ejemplo, usariamos una combinacion de los dos ultimos metodos ">" y "+", ejemplo header>nav+p+a

EJEMPLO: header>nav+p+a daria el siguiente codigo (sin los puntos)
< header>
....< nav>< /nav>
....< p>< /p>
....< a href="">< /a>
< /header>


(header>div^p) Si queremos que "header" contenga la etiqueta "nav" pero que la etqueta "p" este fuera de "header":

escribiriamos "header>div^p" esto hara que "div" este dentro de "header", pero la etiqueta "p" quedaria fuera de "header"

EJEMPLO (sin los puntos):
< header>
....< div>< /div>
< /header>
< p>< /p>

Ahora bien, si hay mas etiquetas no bastaria con un solo "^". Por ejemplo si "header" contiene a "nav" que a su vez contiene"div" y queremos poner la etiqueta "p" fuera del header, no bastaria con un solo "^" necesitariamos uno para salir de la etiqueta "nav" y otro para salir de la etiqueta "header" por lo cuar se escribiria asi "header>nav>div^^p"

EJEMPLO (sin los puntos):
< header>
....< nav>
........< div>< /div>
....< /nav>
< /header>
< p>< /p>


(p{hola}*9)agregar texto a una etiqueta y multiplicar la etiqueta

para hacer esto, escribimos el nombre de la etiqueta y entre llaves pondremos el texto que contendra esa etiqueta, ademas podemos multiplicar el numero de veces que se repetita esa etiqueta agregando "*" + el numero de veces que se desea repetir

EJEMPLO:
p{hola}*3
< p>hola< /p>
< p>hola< /p>
< p>hola< /p>


(a[href="#"])AGREGAR ATRIBUTOS A LAS ETIQUETAS

Los atributos se agregan dentro de los corchetes "[]" y si ademas queremos agregar texto lo hariamos asi "a[href="#"]{aqui va el texto}"

EJEMPLO:
a[href="#"]{texto aqui}
< a href="#">texto aqui< /a>


COMO CREAR UN "DIV" CON UN ATRIBUTO, CLASE, HREF, ETC

Para hacerlo solo necesitamos poner un punto seguido del tipo de dato como hemos visto anteriormente, por ejemplo ".[href="#"]" creara un "div" con "href". Tambien serviria para crear por ejemplo un div con un contenedor asi ".contenedor"

EJEMPLO:
.[href="#"]
< div class="" href="#">< /div>
EJEMPLO2:
.contenedor < div class="contenedor">< /div>


Conteo automatico, SI QUEREMOS HACER POR EJEMPLO ETIQUETAS "P" QUE CUENTEN DEL 1 AL 10---"p{$}*10"

pondremos el nombre de la etiqueta (ya sea nav, p, header, etc) seguido de llaves "{}" que contendran el signo pesos asi "{$}", luego fuera de las llaves "*" + "numero de veces que queremos que cuente". esto serviria por ejemplo para crear varias clases,

ejemplo: p{parrafo numero $}*5 daria como resultado el siguiente codigo:
< p>parrafo numero 1< /p>
< p>parrafo numero 2< /p>
< p>parrafo numero 3< /p>
< p>parrafo numero 4< /p>
< p>parrafo numero 5< /p>

Si por ejemplo queremos hacer varias etiquetas con diferentes clases, lo hariamos asi "a.clase$*4" esto crearia 4 etiquetas "a" con si respectiva clase (clase1, clase2, clase3, clase4)). Este codigo daria el siguente resultado:


< a href="" class="clase1">< /a>
< a href="" class="clase2">< /a>
< a href="" class="clase3">< /a>
< a href="" class="clase4">< /a>

< a href="" class="clase1">< /a>
< a href="" class="clase2">< /a>
< a href="" class="clase3">< /a>
< a href="" class="clase4">< /a>


Si ademas quisieramos crear una etiqueta con items enumerados del 1 al 5 lo hariamos asi "li{item #$}*5" y tendriamos el siguiente resultado

< li>item #1< /li>
< li>item #2< /li>
< li>item #3< /li>
< li>item #4< /li>
< li>item #5< /li>


COMO AGRUPAR ELEMENTOS "header>(nav>ul>li{hola}*3)+p"

esa linea diria que "p" sera hermano de todo lo que este dentro d elos parentesis(nav>ul>li{hola}*5) y ambos estaran dentro de la etiqueta "header"

ejemplo:
"header>(nav>ul>li{hola}*5)+p" dara como resultado el siguiente codigo (SIN LOS PUNTOS)
< header>
....< nav>
........< ul>
............< li>hola< /li>
............< li>hola< /li>
............< li>hola< /li>
............< li>hola< /li>
............< li>hola< /li>
........< /ul>
....< /nav>
< /header>

Tambien podemos agrupar 2 conjuntos de la siguiente manera "header>(nav>ul>li{hola}*3)+(div>p+a)"

EJEMPLO:
"header>(nav>ul>li{hola}*3)+(div>p+a)" daria el siguiente codigo (SIN LOS PUNTOS)
< header>
....< nav>
........< ul>
............< li>hola< /li>
............< li>hola< /li>
............< li>hola< /li>
........< /ul>
....< /nav>
....< div>
........< p>< /p>
........< a href="">< /a>
....< /div>
< /header>
